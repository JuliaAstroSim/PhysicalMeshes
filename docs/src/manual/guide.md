# Guide

## Installation

From the Julia REPL, type `]` to enter the Pkg REPL mode and run
```julia
pkg> add PhysicalMeshes
```
or add from git repository
```julia
pkg> add https://github.com/JuliaAstroSim/PhysicalMeshes.jl
```

Test the package by
```julia
pkg> test PhysicalMeshes
```
